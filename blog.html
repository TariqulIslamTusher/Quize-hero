What does you mean by Event loop?

Event loop is a javaScript runtiome model which is responsible for executing the code, collecting and proccesing the events and executing queued sub-tasks. It is the secretes behing javaScript's asynchornous programming. JavaScript execute all operations in a sigle thread, but using a few data structure it gives the illusion of multi-threading experience. In imagination Event loop is held by a call stack , a event queue and a broser api. Event queue sends the operators or functions in the stack on fill basis, when the stack is empty the event queue again sent the function maintaining the sequence of the code. Whenever an async function is called, it is sent to a browser API. These are APIs built into the browser. Here the browser api sent as a seperate threads. It runs the async function in the single thread. As example 'setTimeOut' method. when a 'setTimeOut' methode run into a stack, it sends to its corresponding API's and wait till the specified time to send the function back in the stack to oparate again.
